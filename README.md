# Enterprise Knowledge Graph Extraction Benchmark (KGEB)

KGEB is an open-source reproducible benchmark to evaluate entity and relation extraction from semi-structured enterprise text records. It provides a baseline, evaluation pipeline, and testing harness for research and practitioner workflows.

## Quickstart
1. Install dependencies:
   - Linux/macOS: scripts/setup.sh
   - Windows (cmd): python -m pip install -r requirements.txt

2. Run extraction and evaluation (Linux):
   - ./scripts/run_test.sh

3. Run on Windows (cmd):
   - scripts\\run_test.bat
